# 🧾 Penetration Testing Portfolio

Cybersecurity enthusiast passionate about penetration testing, vulnerability analysis, and offensive security, with a commitment to continuous learning and hands-on experience in identifying and addressing security risks. 

## 📌 This repository serves as:

* A structured portfolio of penetration testing writeups
* Documentation of my offensive security learning journey as I work towards the OSCP certification
* Evidence of practical, hands-on cybersecurity skills
* A demonstration of OSCP-aligned methodology and thinking

## 🧭 Methodology

All assessments and labs in this repository follow a structured penetration testing approach:

1. Reconnaissance & Enumeration
2. Vulnerability Identification
3. Exploitation
4. Privilege Escalation
5. Post-Exploitation Analysis
6. Remediation Recommendations

This aligns with OSCP-style methodology and real-world penetration testing workflows.

## ⚠️ Disclaimer

All work in this repository is performed in legal, controlled environments such as:

* Hack The Box
* OffSec Proving Grounds

No unauthorised testing has been performed on real-world systems.
