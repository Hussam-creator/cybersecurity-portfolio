
Allows file system access over a network as if they were local, same purpose as SMB. NFS usually runs on ports 111 and 2049

If we discover an NFS service with nmap we can mount it on our machine. We create an empty NFS share to mount it to, then we can navigate the file system

Show available shares
```
showmount -e 10.129.14.128
```

Mounting
```
mkdir target-NFS
sudo mount -t nfs 10.129.14.128:/ ./target-NFS/ -o nolock
cd target-NFS
tree .
```

Listing Contents with username and UIDS
```
ls -l mnt/nfs/
ls -n mnt/nfs/
```

Unmounting
```
cd ..
sudo mount ./target-NFS
```
