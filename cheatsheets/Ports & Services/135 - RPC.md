# TCP 135 (RPC)

## Overview

During Windows enumeration, port 135 is commonly associated with Microsoft Remote Procedure Call (MSRPC), which is used for communication between Windows services. It often acts as an entry point for further enumeration of DCOM services and can help identify additional attack surfaces on Windows hosts. When exposed internally, it is typically used alongside other Windows services such as SMB.

---
```
rpcclient [target IP] -U '' -N
```

With Authentication
```
rpcclient -U [username] [target IP]
```

```
rpcclient [target IP] -U '' -N -c "enumdomusers"
```
```
rpcclient [target IP] -U '' -N -c "enumdomgroups"
```
```
rpcclient [target IP] -U '' -N -c "querydispinfo"
```
